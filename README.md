## 💻 Installation & Setup

This application is a **static site** requiring no server or database.

### Option A: Direct Usage (Recommended)
1.  Download the `index.html` file.
2.  Double-click to open it in your browser.

### Option B: Deploy with GitHub Pages
1.  Go to your repository settings.
2.  Navigate to the **"Pages"** tab.
3.  Under **Branch**, select `main` (or `master`) and the `/ (root)` folder.
4.  Click **Save**. Your app will be live in a few minutes at `https://yourusername.github.io/your-repo/`.

### Option C: Run via Local Server (VS Code)
If you are using VS Code, install the "Live Server" extension:
1.  Right-click `index.html`.
2.  Select **"Open with Live Server"**.
